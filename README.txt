To open the TO DO App:
- download the repo
- double click on the index.html file

To play the TO DO App:
- input a task that must be completed in the input task box
- if it is an important task, click the star
- press add task to append the task to the list
- if you have completed the task, click on it to check it off
- if you want to clear the task from the list, press the x on the right side of the row
- if you do not input a task and press the add task button, an alert will appear
